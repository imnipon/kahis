6. FIELD MAPPING LOGIC
-----------------------
display_name: SA-PDT preferred term, then first SA-PDT term, then VSCT preferred/FSN, then SCT International, then Concept ID.
sapdt_* fields: populated from the current SA-PDT Description or Concept data; blank for VSCT-only rows where no SA-PDT exists.
snomed_* fields: populated from VSCT first, then filled field-by-field from SCT International when VSCT has no value.
match_type: Concept ID match when a SNOMED Concept ID is available; otherwise No match.
match_confidence: Exact for Concept ID match; None when no SNOMED ID is available.
synonym: current row term plus source synonyms and approved searchable synonym_alias values.
